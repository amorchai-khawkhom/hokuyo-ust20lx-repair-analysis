# Hokuyo UST-20LX Repair and Internal Analysis

Industrial LiDAR teardown and repair investigation of the Hokuyo UST-20LX sensor.

This repository documents:
- Internal hardware inspection
- Optical subsystem teardown
- PCB inspection
- Mechanical structure
- Laser optics observations
- Embedded electronics analysis

---

# Device Information

- Model: Hokuyo UST-20LX
- Type: Industrial 2D LiDAR
- Manufacturer: Hokuyo Automatic Co., Ltd.
- Year: 2018

- Rotating optical assembly
- Laser emitter
- Embedded processing board
